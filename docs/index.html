about

GHOST PROVIDER automates and simplifies working with localhost. Analyze your system, discover running services, deploy any GitHub repo as a Docker container, and manage everything from a beautiful terminal interface — no manual config needed.

Paste a GitHub URL and get an instant host score (0–100). If it's deployable, GhostProvider handles the build, port mapping, and container lifecycle. Supports Python, Node.js, Go, Rust, static HTML, and Docker Compose projects.

Built for self-hosting. No cloud, no vendor lock, no subscriptions — just your localhost and Docker.
